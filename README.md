# PlanIt Project

✅ **기능**
- 회원가입 / ID찾기 / 비밀번호 찾기
- 이메일 인증
- 로그인
- 일정 등록 / 수정 / 삭제 (하루 단위 체크리스트)
- 당일 일정 알람 발송

💡 **기술 스택**
- Backend : Java 17, Spring Boot 3, Spring Security, JPA (Hibernate), MySQL, JavaMailSender, WebSocket (STOMP)
- Frontend : React (미완성)
